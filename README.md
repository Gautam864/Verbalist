# Verbalist

Verbalist is a voice-to-list application designed to simplify task management and note-taking. With its intuitive interface and AI-powered audio processing, Verbalist transforms your spoken words into organized, editable lists.

## Features

- **Voice Memo Recording**: Capture voice memos with a single tap.
- **AI-Powered Processing**: Convert voice memos into clear, concise text, removing conversational fillers.
- **List Management**: Create, edit, and organize lists effortlessly.
- **Task Completion**: Mark items as complete with visual feedback.
- **User-Friendly Design**: Accessible layout with large icons and spacious tap targets.

## Tech Stack

- **Frontend**: React, Next.js, TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Custom-built components with accessibility in mind
- **AI Integration**: Custom AI utilities for voice-to-text processing
- **Build Tools**: PostCSS, ESBuild
- **Version Control**: Git
- **Package Management**: npm

## Getting Started

1. Clone the repository.
2. Enter your Gemini API Key (Available for free)
3. Install dependencies using `npm install`.
4. Start the development server with `npm run dev`.
